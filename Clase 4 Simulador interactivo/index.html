Ejercicio

Dado un String, verificar si es un palíndromo [Una cadena que no cambia cuando se invierte (lee lo mismo hacia atrás y hacia adelante)].

Instrucciones

Este programa recibe un string y verifica si este es un políndromo [Una cadena que no cambia cuando se invierte (lee lo mismo hacia atrás y hacia adelante)], aquí dejo algunos ejemplos para que pruebes (aba, Oso, rajar, oro, rapar, rayar), el programa finalizará cuando escribas la palabra [fin]
